# LP CONSULTORIA | Direcionamento Estratégico

Landing page profissional para consultoria de Direcionamento Estratégico em Vida e Negócios.

## 🚀 Sobre o Projeto

Landing page de alta conversão desenvolvida com:
- **HTML5** semântico
- **TailwindCSS** via CDN
- **Space Grotesk** e **Inter** (Google Fonts)
- **Material Symbols** para ícones
- Design system **Architectural Monolith** (dark mode)

## 🎨 Design System

- **Paleta**: Laranja queimado (#FF6B00) + Azul elétrico (#ADC7FF) sobre fundo escuro (#131313)
- **Tipografia**: Space Grotesk (headlines) + Inter (body)
- **Bordas**: Minimalistas, estilo arquitetural
- **Efeitos**: Glassmorphism, reveal-on-scroll, hover transitions
